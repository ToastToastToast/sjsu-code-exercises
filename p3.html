Exercise 5

My bit project has a total of 11 different javascript functions to display the 5 arrays. This exercise was a group exercise so 3 of the arrays are created by group members listed in the sketch, Irvana Gottwald and Aakash Kumar. Nested for loops call on either numbers or text in an array and display different visuals depending on what variable is being called. For example, the number 4 calls a dark orange color that is displayed as a square or a pixel. The image arrays use minecraft images to display the first 2 arrays depicing a minecraft wolf and a Labrador. The arrays are brought together to create two scenes. The first being a haunted scene with the game AmongUs character chasing Link from Nintendo. The second scene is the population of dogs in a heavenly cloud environment praying to the chicken.
